Nested intervals

Theorem 7 :

For closed interval \(I_n = [a_n,b_n](n=1,2,...)\),

  1. Every interval \(I_n\) is included in \(I_{n+1}\),
  2. With infinitely increasing \(n\), \(b_n - a_n\) diminishes without limit,
there exists only one point that is common with every \(I_n\).

It is named Nested interval to confirm a number using this theorem.

From first assumption, \begin{equation} a_1 \le a_2 \le ... \le a_m \le ... \le b_n \le ... \le b_2 \le b_1 . \end{equation} Therefore, both \(\{a_n\}\) and \(\{b_n\}\) are monotonic, and are bounded. Then, from theorem 6, there exists \(\alpha , \beta \) such that \begin{equation} \lim a_n = \alpha , \lim b_n = \beta . \end{equation} Because \(a_n < b_n\) for arbitrary \(m,n\), when \(n \to \infty \), \(\alpha < b_m\). Hence when \(m \to \infty\), \(\alpha \le \beta\).

From second assumption, for arbitrary \(\epsilon > 0\), there exist \(n\) such that \begin{equation} b_n - a_n < \epsilon . \end{equation} From \begin{equation} a_n \le \alpha \le \beta \le b_n , \end{equation} we can rewrite it as \begin{equation} 0 \le \beta - \alpha < \epsilon . \end{equation} \(epsilon\) is arbitrary chosen, hence \(\alpha = \beta\).
